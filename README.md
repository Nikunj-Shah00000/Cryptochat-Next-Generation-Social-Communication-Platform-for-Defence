## 🚀 New Features – Cryptochat (Secure Defense Messenger)

Cryptochat continues to evolve with powerful, security-first enhancements designed for mission-critical communication environments.

### 📦 Real-Time Backup System

* **Auto-backup every 5 minutes** for continuous data protection
* **Manual snapshots** for on-demand backups
* **Backup history with timestamps** for easy tracking
* **Full data coverage** – messages, contacts, and settings
* **One-click restore** from any previous state

* Create **custom contacts** with name, rank, agency, and military ID

* Designed for structured, defense-grade identity handling

* Streamlined **sidebar actions** (Add, Backup, Restore)
